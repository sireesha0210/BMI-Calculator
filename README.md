## Description
This project allows users to calculate their BMI and classify it into standard health categories.  

- **Beginner Version:** Command-line interface
- **Advanced Version:** GUI interface with Tkinter, data storage, and BMI trend visualization

---

## Features

### Beginner Version:
- Calculate BMI based on weight (kg) and height (m)
- Classify BMI into categories: Underweight, Normal, Overweight, Obese
- Input validation and error handling

### Advanced Version:
- Graphical User Interface (Tkinter)
- Store BMI data in CSV file
- Display historical BMI trends with Matplotlib graphs
- Responsive and user-friendly interface
- Handles invalid inputs and file errors gracefully

---

## BMI Categories

| BMI Value        | Category         |
|-----------------|-----------------|
| < 18.5          | Underweight     |
| 18.5 – 24.9     | Normal weight   |
| 25 – 29.9       | Overweight      |
| ≥ 30            | Obese           |
